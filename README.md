# arbinex_bottom_bar

A customizable Flutter bottom bar package with a center action button.

## Features

- Custom bottom bar items with icon and label
- Separate center action slot
- Controlled and uncontrolled active index support
- Custom colors, shadows, spacing, and sizing

## Usage

```dart
import 'package:arbinex_bottom_bar/arbinex_bottom_bar.dart';
```

See the `/example` app for a full usage sample.
